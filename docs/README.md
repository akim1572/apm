# A(kim's) P(assword) M(anager)

This is a password manager written in C. Being able to store in a file and
to not only store but generate passwords, delete, name, as well as search them
efficiently.

## Installation

Be sure to have CMake installed to install properly
```bash
cmake -S ./
cmake -B bin/
make
```
## Usage
Use this command to learn all the others.
```bash
./apm -h
```
